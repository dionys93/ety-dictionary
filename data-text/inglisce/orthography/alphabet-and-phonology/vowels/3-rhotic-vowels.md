## 3. The Rhotic Vowels (R-Controlled)

| Grapheme | IPA | Environment / Context | Inglisce Examples | Modern English |
| :--- | :--- | :--- | :--- | :--- |
| **AÎR / EÎR** | /ɛr/ | Stressed derivatives | comp*aîr*able, unb*eîr*able | comparable, unbearable |
| **AIR-E** | /ɛr/ | Default | c̃*aire*, h*aire*, c*aire* | chair, hair, care |
| **AR** | /ɑr/ | Default stressed (monosyllables and closed syllables) | c*ar*re, st*ar*re, p*ar*meșan | car, star, parmesan |
| **AR** | /ɚ/ | Word-final nouns and comparatives | eu*ar*s, big*ar*, laûd*ar* | ewers, bigger, louder |
| **ÁR** | /ɑr/ | Stress-marked | al*ár*ms, commiss*ár*s | alarms, commissars |
| **ÂRE** | /ɛr/ | Latin/French roots replacing -ARE | squ*âre*, prep*âre* | square, prepare |
| **AUR** | /ɔr/ | Specific lexical set | r*aur*e, b*aur*e, p*aur* | roar, boar, poor |
| **ER** | /ɚ/ | Infinitive verb ending | mast*er*, slomb*er*, rememb*er*, <br> splat*er*, flat*er*, uát*er*, þond*er* | master, slumber, remember, <br> splatter, flatter, water, thunder |
| **ERE** | /ɚ/ | Infinitive verb/noun ending | ham*ere*, ans*ere*, <br> stam*ere*, man*ere* | hammer, answer, <br> stammer, manner |
| **EARE** | /ɛr/ | Word-final specific | b*eare*, p*eare*, s*eare* | bear, pear, share |
| **EIRE** | /ɛr/ | Specific verbs/words | b*eire*, u*eire*, t*eire* | bear, wear, tear |
| **EUR** | /ɚ/ | Default agent nouns | tiec̃*eur*, peint*eur* | teacher, painter |
| **Ê** | /ɚ/ | Stressed marked | s*ê*ve, *ê*ne, u*ê*me, pref*ê*r | serve, earn, worm, prefer |
| **ÈR** | /ɛr/ | Marked stress | *èr*a, b*èr*ie | era, bury |
| **IAR-E** | /ɪər/ | Historical 'ear' lexical set | d*iare*, f*iare*, y*iare*, *iar*s | dear, fear, year, ears |
| **IAR** | /iɚ/ | Comparatives ending in -y | sill*iar* | sillier |
| **IER-E** | /ɪər/ | Historical 'eer', 'ere', and 'ier' sets | n*ier*, h*iere*, c̃*iere*, b*iere*, <br> d*iere*, p*iere*, t*iere*, st*iere* | near, here, cheer, beer, <br> deer, pier, tier, steer |
| **IOR** | /ɪər/ | 'eer' words | st*ior*e | steer (animal) |
| **IOR** | /ɚ/ | Latin derivatives | amat*ior*, criet*ior* | amateur, creature |
| **IR** | /ɚ/ | Stressed closed syllables and unstressed suffixes | b*ir*de, u*ir*c, u*ir*se, <br> mat*ir*, cov*ir*, zef*ir*s | bird, work, worse, <br> matter, cover, zephyrs |
| **IRE** | /ɪər/ | Noun and verb alternatives | volont*ire*, gu*ire*, n*ire* | volunteer, gear, near (verb) |
| **ÎAR / ŶAR** | /aɪər/ | Noun plurals | co*îar*s, f*ŷar*s, des*îar*s | choirs, fires, desires |
| **ÎER** | /aɪər/ | Verb infinitive & progressive | acqu*îer*, acqu*îer*ing | acquire, acquiring |
| **ÎRE** | /aɪər/ | Default for nouns and verb present tense | des*îre*, fu*îre*, acqu*îre* | desire, fire, acquire |
| **OR-E** | /ɔr/ | Inherently stressed / Monosyllables | c*or*ne, f*or*me, n*or*þ, <br> ign*ore*, abh*ore* | corn, form, north, <br> ignore, abhor |
| **OR** | /ɚ/ | Unstressed Medial / Agent Nouns | c*or*ve, uirc*or* | curve, worker |
| **ÓR** | /ɔr/ | Stressed medial before Latinate suffixes | man*ór*ial, hist*ór*ical | manorial, historical |
| **OUR-E** | /ɔr/ | Select words | fl*oure*, d*oure*, f*our* | floor, door, four |
| **OUR** | /ɚ/ | General medial | m*our*der, n*our*se, t*our*ne | murder, nurse, turn |
| **RE** | /ɚ/ | Noun terminals | fáþ*re*, riv*re*, mat*re*, eu*re*, <br> zef*re*, nev*re*, forev*re*, aft*re* | father, river, matter, ewer, <br> zephyr, never, forever, after |
| **UAR** | /ɔr/ | Following U | *uar*m, *uar*ne, s*uar*me, g*uar*e | warm, warn, swarm, war |
| **URE** | /ɚ/ | Unstressed Latin -ura derivatives (often after affricates) | procíḑ*ure*, meș*ure*, litt*ure*, scelt*ure* | procedure, measure, litter, shelter |
| **URE** | /jʊər/ | Stressed Yod class (word-final open) | c*ure*, p*ure*, sec*ure* | cure, pure, secure |
| **ÚR** | /jʊər/ | Stressed Yod class (before consonants or suffixes) | c*úr*s, p*úr*ifae | cures, purify |
| **ÜAR** | /ɔr/ | Following Ü (explicit glide) | a*üar*d, re*üar*d | award, reward |
| **YRE** | /ɪər/ | Select words | h*yre*, *yre* | hear, ear |
| **ŶRE** | /aɪər/ | Select words | f*ŷre* | fire |
