## 3. The Rhotic Vowels (R-Controlled)

| Grapheme | IPA | Environment / Context | Inglisce Examples | Modern English |
| :--- | :--- | :--- | :--- | :--- |
| **AÎR / EÎR** | /ɛr/ | Stressed derivatives | comp*aîr*able, unb*eîr*able | comparable, unbearable |
| **AIR-E** | /ɛr/ | Default | c̃*aire*, h*aire*, c*aire* | chair, hair, care |
| **AR** | /ɑr/ | | c*ar*re, st*ar*re, p*ar*meșan | car, star, parmesan |
| **AR** | /ɚ/ | Word-final nouns and comparatives | eu*ar*s, big*ar*, laûd*ar* | ewers, bigger, louder |
| **ÁR** | /ɑr/ | Stress-marked | al*ár*ms, commiss*ár*s | alarms, commissars |
| **ÂRE** | /ɛr/ | Latin/French roots replacing -ARE | squ*âre*, prep*âre* | square, prepare |
| **AUR** | /ɔr/ | Specific lexical set | r*aur*e, b*aur*e, p*aur* | roar, boar, poor |
| **EARE** | /ɛr/ | Word-final specific | b*eare*, p*eare*, s*eare* | bear, pear, share |
| **EIRE** | /ɛr/ | Specific verbs/words | b*eire*, u*eire*, t*eire* | bear, wear, tear |
| **EUR** | /ɚ/ | Default agent nouns | tiec̃*eur*, peint*eur* | teacher, painter |
| **Ê** | /ɚ/ | Stressed marked | s*ê*ve, *ê*ne, u*ê*me | serve, earn, worm |
| **ÈR** | /ɛr/ | Marked stress | *èr*a, b*èr*ie | era, bury |
| **IAR-E** | /ɪər/ | | d*iare*, f*iare*, gu*iare* | dear, fear, gear |
| **IAR** | /iɚ/ | Comparatives ending in -y | sill*iar* | sillier |
| **IER-E** | /ɪər/ | | n*ier*, h*iere*, c̃*iere*, b*iere*, d*iere*, p*iere*, t*iere*, st*iere* | near, here, cheer, beer, deer, pier, tier, steer |
| **IOR** | /ɪər/ | 'eer' words | st*ior*e | steer (animal) |
| **IOR** | /ɚ/ | Latin derivatives | amat*ior*, criet*ior* | amateur, creature |
| **IR** | /ɚ/ | General medial | b*ir*de, u*ir*c, u*ir*se, mat*ir*, cov*ir*, zef*ir*s | bird, work, worse, matter, cover |
| **IRE** | /ɪər/ | Noun and verb alternatives | volont*ire*, n*ire* | volunteer, near (verb) |
| **ÎAR** | /aɪər/ | Noun plurals | co*îar*s, fu*îar*s, des*îar*s | choirs, fires, desires |
| **ÎER** | /aɪər/ | Verb infinitive & progressive | acqu*îer*, acqu*îer*ing | acquire, acquiring |
| **ÎRE** | /aɪər/ | Default for nouns and verb present tense | des*îre*, fu*îre*, acqu*îre* | desire, fire, acquire |
| **OR-E** | /ɔr/ | Inherently stressed / Monosyllables | c*or*ne, f*or*me, n*or*þ, ign*ore*, abh*ore* | corn, form, north, ignore, abhor |
| **OR** | /ɚ/ | Unstressed Medial / Agent Nouns | c*or*ve, uirc*or* | curve, worker |
| **OUR-E** | /ɔr/ | Select words | fl*oure*, d*oure*, f*our*  | floor, door, four  |
| **OUR** | /ɚ/ | General medial | m*our*der, n*our*se, t*our*ne | murder, nurse, turn |
| **RE** | /ɚ/ | Noun terminals | fáþ*re*, riv*re*, mat*re*, eu*re*, zef*re* | father, river, matter, ewer, zephyr |
| **UAR** | /ɔr/ | Following U | *uar*m, *uar*ne, s*uar*me, g*uar*e | warm, warn, swarm, war |
| **URE** | /ɚ/ | Latin -ura derivatives | procíḑ*ure*, meș*ure* | procedure, measure |
| **ÜAR** | /ɔr/ | Following Ü (explicit glide) | a*üar*d, re*üar*d | award, reward |
| **YRE** | /ɪər/ | Select words | h*yre* | hear |
