# IPA /ɛ/ — "BED" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /ɛ/ |
| Description | Mid-front unrounded vowel |
| Example in Modern English | bed, net, dress |
| Frequency | Very common; one of the primary stressed vowel sounds |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | E, E-E |
| Stress-marked | È |
| Alternative | AI |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

---

## Core Distribution Rules

### 1. E / E-E = /ɛ/ (Most Common)

The default spelling for /ɛ/ in stressed syllables.

| Modern | Inglisce |
|--------|----------|
| net | nete |
| bed | bede |
| dress | * |
| repellent | * |
| head | hede |
| stead | stede |
| to wet | to ouete |
| to set | to sete |
| to rebel | to rebelle |
| to inspect | to inspecte |
| elephant | elefant |
| elevator | elevaitor |

---

### 2. È = /ɛ/ (Irregular Stress)

Grave accent marks stressed /ɛ/ where stress is not predictable from spelling alone.

| Modern | Inglisce |
|--------|----------|
| anemone | anèmonie |
| imperative | impèratif |
| president | prèsident |

---

### 3. AI = /ɛ/ (Alternative Spelling)

Used in certain high-frequency words and past tense forms corresponding to Modern English EA, ET, and short E.

#### Common Words

| Modern | Inglisce |
|--------|----------|
| lettuce | laitace |
| to catch | to caic̃e |

#### Past Tense Forms

| Modern | Inglisce |
|--------|----------|
| let (past) | lait |
| read (past) | raid |
| to get | to gait |
