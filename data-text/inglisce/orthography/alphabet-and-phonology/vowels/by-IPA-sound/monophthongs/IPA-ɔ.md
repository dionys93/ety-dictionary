# IPA /ɔ/ — "THOUGHT" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /ɔ/ |
| Description | Mid-back rounded vowel |
| Example in Modern English | thought, caught, lawn |
| Frequency | Moderately common; several spelling patterns |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | AU |
| Word-final | -AHE |
| Silent GH clusters | -OH, -AUH |
| Stress-marked | Ò |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

---

## Core Distribution Rules

### 1. AU = /ɔ/

The default spelling for /ɔ/, corresponding to Modern English AU, AW, and AL.

| Modern | Inglisce |
|--------|----------|
| autumn | automn |
| automatic | * |
| lawn | laone |
| dawn | daone |
| pawn | paone |
| coffee | caufie |
| Australia | Austrâlia |

---

### 2. -AHE / -AUS = /ɔ/ (Word-Final)

Used in monosyllables ending in Modern English -AW. Note the irregular plural pattern: -AHE → -AUS.

| Modern | Inglisce |
|--------|----------|
| law, laws | lahe, laus |
| saw, saws | sahe, saus |
| draw, draws | drahe, draus |
| shah, shahs | șahe, șaus |

---

### 3. -ALLE / -OLLE = /ɔ/ (Word-Final -ALL)

Used in monosyllables ending in Modern English -ALL or -AWL.

#### -ALLE (from -AWL and some -ALL)

| Modern | Inglisce |
|--------|----------|
| all | * |
| fall | falle |
| crawl | cralle |
| drawl | dralle |
| stall, stalls | stalle, stalls |

#### -OLLE (from -ALL with rounded vowel)

| Modern | Inglisce |
|--------|----------|
| ball, balls | bolle, bolls |
| doll, dolls | dolle, dolls |

---

### 4. -OH / -AUH = /ɔ/ (Silent GH Clusters)

Used where Modern English retains a silent GH. -OH derives from OUG(H); -AUH from AUG(H)/OUG(H).

| Modern | Inglisce |
|--------|----------|
| bought | boht |
| fought | foht |
| sought | soht |
| thought | þoht |
| cough | cohfe |
| caught | cauht |
| taught | tauht |

---

### 5. Ò = /ɔ/ (Irregular Stress)

Grave accent marks stressed /ɔ/ where stress is not predictable from spelling alone.

| Modern | Inglisce |
|--------|----------|
| cost | còste |
| frost | fròste |
| lost | lòst |
| soft | sòft |
| coffer, coffers | coffre, coffars |
