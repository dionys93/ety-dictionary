# IPA /ɚ/ — "NURSE" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /ɚ/ |
| Description | Mid-central rhotacized vowel |
| Example in Modern English | nurse, bird, earn |
| Frequency | Common; appears in multiple spelling patterns |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Agent nouns | -EUR, -EOR |
| Any | -OUR, -OR-, -IR- |
| Verbs | -ER, -ORE, -ARE |
| Nouns | -RE, -ARS, -ORS, -IRS, -AR |
| Stressed | Ê |
| Comparatives | -AR, -IAR |
| Latin derivatives | -IOR |
| Latin -ura derivatives | -URE |

/ɚ/ has the widest range of spelling realizations of any vowel in Inglisce; distribution is largely determined by word class and etymology
