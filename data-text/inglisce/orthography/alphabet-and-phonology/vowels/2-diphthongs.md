## 2. The Diphthongs (Complex Glides)

| Grapheme | IPA | Environment / Context | Inglisce Examples | Modern English |
| :--- | :--- | :--- | :--- | :--- |
| **AE / ÁI** | /aɪ/ | Semantic contrast (distinguishes homophones) | fr*ae*, fr*ái*s, c̃*ae* l*ái*e, m*ái*t | fry (fish), fries (fish), lie (recline), might (verb) |
| **AI-E** | /eɪ/ | Default for verbs and mid-word nouns | s*aie*, r*ai*gn*e*, d*aie* | say, rain, day |
| **AI** | /aɪ/ | Historical preservation (lexical set) | *aie*le, *aie*ls | aisle, aisles |
| **AIH** | /aɪ/ | Limited verbal roots |n*aih*te, d*aih*e, m*aih*te, m*aih*ty | night, die, might, mighty |
| **AIT** | /eɪ/ | Unstressed Latin verbs | demonstr*ait*, navig*ait* | demonstrate, navigate |
| **AUND** | /aʊ/ | Germanic strong past | f*aund*, gr*aund* | found, ground |
| **AÛ** | /aʊ/ | Open lexical class | h*aû*se, m*aû*se, l*aû*se, cr*aû*de s*aû*re, l*aû*d | house, mouse, louse, crowd, sour, loud |
| **AY** | /eɪ/ | Adjectives, plural nouns, and before suffixes | gr*ay*, d*ay*s, p*ay*ing, s*ay*ing | gray, days, paying, saying |
| **Â** | /eɪ/ | Stressed default / Stressed Latin verbs | m*â*che, g*â*te, rel*â*te| make, gate, relate |
| **EA** | /eɪ/ | Replaces Â-E pattern (often after S to trigger /ʃ/) and historical retention | gr*ea*t, s*ea*de, s*ea*che, s*ea*me | great, shade, shake, shame |
| **EAI** | /aɪ/ | Historical preservation (lexical set) | *eai*e, *eai*s | eye, eyes |
| **EI-E** | /eɪ/ | Alternative spelling | conv*ei*e, r*ei*gne, r*ei*ns | convey, reign, reins |
| **ÉI** | /eɪ/ | Stress-marked | *éi*me, cont*éi*nment | aim, containment |
| **EIH** | /eɪ/ | Replaces historical *-eigh* | u*eih*te, sl*eih*e | weight, sleigh |
| **EY** | /eɪ/ | Word-final pronouns/adjectives, and before suffixes | conv*ey*ing, þ*ey*, ou*ey* | conveying, they, whey |
| **Î** | /aɪ/ | Default representation | n*î*hte, cr*î*e, f*î*nde, sch*î*s | night, cry, find, skys |
| **O-E** | /oʊ/ | Closed syllables, word-final open, and multisyllables | b*o*t*e*, b*o*n*e*, st*o*n*e*, c*o*d*e*, uind*o*, se*oe*, bl*oe*, al*o*n*e*, my*o*n*e* | boat, bone, stone, code, window, show, blow, alone, my own |
| **Ó** | /oʊ/ | Marked stress | hipn*ó*sis, b*ó*ste**, c*ó*ld | hypnosis, boast, cold |
| **OA** | /oʊ/ | Lexical "oa" set | b*oa*ts, b*oa*ns, st*oa*ns, c*oa*ste, c*oa*me, c*oa*ms, h*oa*me, h*oa*ms | boats, bones, stones, coast, comb, combs, home, homes |
| **OI-E** | /ɔɪ/ | Medial and word-final | c*oi*ne, c*oi*ns, c̃*oi*ce, enj*oi*e, b*oi*e | coin, coins, choice, enjoy, boy |
| **OIH** | /aɪ/ | Historical *-igh* class & open syllables | h*oih*, fl*oih*e, sc*oih*e, se*oih* | high, fly, sky, shy |
| **OY** | /ɔɪ/ | Before suffixes or vowels (plurals, derived forms) | b*oy*s, t*oy*s, sp*oy*els, ann*oy*ing, ann*oy*ance, destr*oy*eur | boys, toys, spoils, annoying, annoyance, destroyer |
| **Ô** | /aʊ/ | Core default (open final, before vowel, L, or N) | all*ôe*, all*ôu*ing c*ôe*, c*ôu*s, v*ôl*e, v*ôe*ls, *ôl*e, *ôe*ls, s*ôn*de, t*ôu*re | allow, allowing, cow, cows, vowel, vowels, owl, owls, sound, tower |
| **YH** | /aɪ/ | Verbal / participial glide | d*yh*e, cr*yh*ing | dye, crying |
| **Ŷ** | /aɪ/ | Medial before clusters | c*ŷ*nd, c*ŷ*te | kind, kite |
