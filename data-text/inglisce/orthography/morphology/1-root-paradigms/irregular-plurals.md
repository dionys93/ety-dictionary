# Irregular Plurals in Inglisce

While standard nouns in Inglisce form their plurals predictably, a small subset of core Germanic vocabulary retains historical, irregular plural forms. These mostly involve internal vowel mutations (umlaut) or archaic zero-plurals.

### 1. Vowel Mutation Plurals
These nouns change their internal root vowel rather than taking a standard plural suffix.

| Singular, Plural | Modern English |
| :--- | :--- |
| **fûte, fyte** | foot / feet |
| **touþ, tieþ** | tooth / teeth |
| **maûse, mîce** | mouse / mice |
| **laûse, lîce** | louse / lice |
| **dîe, dîce** | die / dice |
| **gouss, guiess** | goose / geese |
| **man, men** | man / men |
| **uôman, uimmen** | woman / women |

### 2. Irregular Suffixes
These nouns take historical Germanic endings rather than standard sibilants.

| Singular, Plural | Modern English |
| :--- | :--- |
| **chîld, c̃ildren** | child, children |
| **ox, oxen** | ox, oxen |

### 3. Zero-Plurals (Unchanged)
These nouns, typically associated with animals or specific categories, have identical singular and plural forms.

| Singular and Plural | Modern English |
| :--- | :--- |
| **diere** | deer |
| **siepe** | sheep |
| **șrimpe** | shrimp |
| **bîsson** | bison |
| **bufalo** | buffalo |

## 4. Etymological Fossils

While the words in this category pluralize using standard Inglisce mechanical reductions (e.g., dropping a terminal `-e` or reducing a heavy cluster to add `-s`), their base singular spellings are highly irregular. 

These words preserve unique historical vowel clusters or archaic consonants to distinguish them visually from homophones, or to reflect deep etymological roots that resist standardization.

| Singular, Plural | Modern English | Etymology |
| :--- | :--- | :--- |
| **eaie, eais** | eye / eyes | From Old English *ēage*, ultimately from Proto-Germanic *\*augô*. |
| **saie, saís** | sea / seas | From Old English *sǣ*, ultimately from Proto-Germanic *\*saiwiz*. |
| **aigue, aigs** | egg / eggs | From Old Norse *egg* (which supplied the hard `/g/`), superseding the native Old English cognate *ǣg*. |
| **aiele, aiels** | aisle / aisles | From Old French *ele* (wing), from Latin *ala*. The silent 's' in Modern English was a later mistake caused by false association with *isle*. |
| **eale** | -- | ale | From Old English *ealu*, ultimately from Proto-Germanic *\*alu*. |
| **fimelle, fimels** | female / females | From Old French *femelle*, from the Latin diminutive *femella*. The Modern English spelling "female" was altered by a false association with the unrelated word *male*. |
