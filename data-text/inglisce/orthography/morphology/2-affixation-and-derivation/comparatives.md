# Comparative and Superlative Adjectives in Inglisce

Inglisce forms comparatives and superlatives using **-ar** and **-est** (derived from Old English *-ra* and *-est*), unlike Modern English's *-er* and *-est*.

## Formacion Reuls

### 1. One-Syllable Adjectives
**Pattern:** Add **-ar** (comparative) and **-est** (superlative).

**Regular Examples:**

| Base | Comparative | Superlative |
| :--- | :--- | :--- |
| **small** | smallar | smallest |
| **tall** | tallar | tallest |
| **oît** | oîtar | oîtest |
| **óld** | óldar | óldest |
| **sad** | sadar | sadest |
| **smart** | smartar | smartest |

**Adjectives Ending in -g:**
When the vowel is long, the ending adjusts:
* *Long vowels + ng:* Add **-u** before the ending.

| Base | Comparative | Superlative |
| :--- | :--- | :--- |
| **stròng** | stròngar | stronguest |
| **lòng** | lòngar | longuest |

* *Short vowels after g:* Add **-h** before the ending.

| Base | Comparative | Superlative |
| :--- | :--- | :--- |
| **big** | bigar | bighest |
| **yong** | yongar | yonghest |

**Adjectives Ending in -c:** Add **-qu** before the superlative ending.

| Base | Comparative | Superlative |
| :--- | :--- | :--- |
| **pinc** | pincar | pinquest |
| **blac** | blacar | blaquest |

---

### 2. Two-Syllable Adjectives
**Pattern:** Default to analytical forms with **more** / **moast**.

**Exception:** Adjectives ending in **-o** can take **-ar** / **-est** with a **-u-** insertion.

| Base | Comparative | Superlative |
| :--- | :--- | :--- |
| **nâro** | nârouar | nârouest |
| **sloe** | slouar | slouest |

---

### 3. Adjectives Ending in -y
**Pattern:** Change **-y** to **-iar** (comparative) and **-iest** (superlative).

| Base | Comparative | Superlative |
| :--- | :--- | :--- |
| **pritty** | prittiar | prittiest |
| **lîvely** | lîveliar | lîveliest |
| **daidly** | daidliar | daidliest |
| **míty** | mítiar | mítiest |
| **ângry** | ângriar | ângriest |

---

### 4. Multi-Syllable Adjectives
**Pattern:** Always use **more** / **moast**.

| Base | Comparative | Superlative |
| :--- | :--- | :--- |
| **sîlent** | more sîlent | moast sîlent |
| **cairful** | more cairful | moast cairful |
| **clevre** | more clevre | moast clevre |
| **sevire** | more sevire | moast sevire |
