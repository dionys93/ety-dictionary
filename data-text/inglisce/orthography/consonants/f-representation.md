# PH / F / FH Representation in Inglisce

The letters **F**, **PH**, and **FH** represent /f/ in Inglisce. Their usage is **layered and positional**, reflecting morphology, contrast preservation, and historical orthography.  

**Diacritics used:** none for this system  
**Related digraphs:** none beyond PH/FH  

---

## 1️⃣ Core Principle

> **Contrast preservation overrides simplification.**  

PH is retained whenever collapsing it to **F** would create a **visual or morphological collision** with an existing F-word.  
This ensures that minimal pairs and distinct stems remain clearly distinguishable.

---

## 2️⃣ Structural Rules

1. **Default /f/** → **f**  
   Most words use **f** for /f/ unless overridden by other rules.  
   - Examples: *farme*, *fîne*, *aftre*

2. **Medial -ph-**  
   All medial occurrences of **-ph-** are preserved.  
   - Examples: *filosophie*, *prophecie*, *symphonie*

3. **PH + R Cluster**  
   **PH** immediately before **R** is replaced by **FR**.  
   - Examples: *phrase → frâse*, *zephyr → zefre*

4. **Initial PH**  
   Collapse **ph- → f-** except when contrast preservation requires retention.  
   - Retained: *phone*, *phonology*, *prophet*  
   - Collapsed: *philosophie → filosophie*, *photograph → fótograph*, *phenomenon → fenomenon*

5. **Historical -GH (/f/) → FH**  
   When -gh represents /f/ in English, it is replaced with **-fh** at the end of a word.  
   - Examples: *rough → rofh*, *tough → tofh*, *enough → enofh*  
   - If followed by a suffix, simplify to **f**: *laughter → lafture*, *toughen → tofane*

---

## 3️⃣ Decision Flow (Compact)

1. Is the word a historical -gh representing /f/?  
   - Yes → apply **-fh** (final) or **f** (before suffix)  
   - No → continue  
2. Is **-ph- medial**?  
   - Yes → keep **ph**  
   - No → continue  
3. Is **ph** before **r**?  
   - Yes → replace with **fr**  
   - No → continue  
4. Is **ph initial**?  
   - Would collapsing create a contrast collision with an existing F-word?  
     - Yes → keep **ph**  
     - No → collapse to **f**  
5. Else → **f**
