# U

The letter **U** is fundamentally a vowel representing **/u/** in Inglisce.

Its value follows four primary structural rules:

U → /u/  
U + consonant → /ə/  
Ú + consonant(s) → /(j)u/  
U + consonant + E → /(j)u/  

The presence of /j/ (yod) depends on the consonant preceding U.

There is no letter **W** in Inglisce. U (and sometimes O) represent the glide /w/ in specific environments.

**Diacritics used:** Ú, Û, Ü  
**Related digraphs:** UE, UI  

'*' Represents no spelling change from Modern (Websterian) English.

---

# I. Yod Classification

The pronunciation of long U (/(j)u/) depends on consonant class.

## Consonants That Trigger Yod (/ju/)

| With Yod | Without Yod |
| --- | --- |
| cu | cou |
| gu | gou |
| pu | pou |
| bu | bou |
| fu, feu | fou |
| hu, heu | hou |
| veu | vou |
| mu | mou |
| -lue | lu-, lou-|
| ju | Always palatalized |

Examples:

| Modern English | Inglisce |
|----------------|----------|
| cute | * |
| muse, music | muse, músic |
| confuse, confusion | confuse, confúsion |

These produce **/ju/** when long.

---

## Consonants Without Yod (/u/)

| Without Yod | With Yod |
| --- | --- |
| tu | țu |
| du | d̦u |
| su | șu |
| nu | ñu |
| zu | șu |
| ru, rou, reu | Always without yod |

Examples:

| Modern English | Inglisce |
|----------------|----------|
| rude | reude |
| rule | reule |

These produce plain **/u/** when long.

---

# II. Closed Syllables

When U is followed by a consonant and not restored by silent -E:

**U = /ə/** (absolute default)

| Modern English | Inglisce | IPA |
|----------------|----------|-----|
| luck | hluc | /lək/ |
| buck | buc | /bək/ |
| run | rone | /rən/ |

---

# III. Closed Syllable Overrides

| Spelling | Value |
|----------|--------|
| U | /ə/ |
| Ú | /(j)u/ |
| Û | /ʊ/ |

Examples:

| Modern English | Inglisce |
|----------------|----------|
| root, roots | rute, rúts |
| fruit, fruits | frute, frúts |
| good | gûd |
| full | fûl |

If silent -e is absent and the vowel must remain /(j)u/, **Ú is required**.

---

# IV. Glide Representation (/w/)

There is no letter **W** in Inglisce.

## U as Glide

U represents /w/ in certain consonant clusters:

- CU
- SU

| Modern English | Inglisce |
|----------------|----------|
| quick | cuic |
| quake | cuâche |
| swerve | suêve |

---

# V. Silent U (Hard G Preservation)

U is silent when preserving hard G before E, I, or Y.

| Modern English | Inglisce |
|----------------|----------|
| give | guive |
| guide | guîde |
| gift | guifte |
| geek | guiec |

---

# VI. M / N / R Aesthetic Rule

When U would represent /ə/ after M, N, or R,  
it is spelled **O** instead.

| Modern English | Inglisce |
|----------------|----------|
| run | rone |
| nun | nonne |
| number | nombre |
| murmur | mormor |
| Monday | * |

---

# VII. Summary

| Pattern | Value |
|----------|--------|
| U | /u/ |
| U + C | /ə/ |
| Ú + C | /(j)u/ |
| U + C + E | /(j)u/ |
| Û | /ʊ/ |
| U (clusters) | /w/ |
| U after G before E/I/Y | silent |
| Ü | /w/ |
