# UI / ÚI

The digraph **UI** represents different sounds depending on stress and position.

---

# I. Pronunciation Overview

| Form | Stress | Value |
|------|--------|--------|
| ÚI | Stressed | /uɪ/ |
| UI | Unstressed | /wi/ or /wɪ/ |
| GUI | — | Hard G + /i/ (U silent) |

---

# II. Stressed ÚI → /uɪ/

When stressed, UI is written **ÚI** and pronounced /uɪ/.

| Modern English | Inglisce |
|----------------|----------|
| to ruin | to rúine |
| the ruin | þe rúin |

This spelling explicitly marks stress and prevents glide reduction.

---

# III. Unstressed UI → /w/ + vowel

When unstressed, **UI** represents a glide /w/ followed by /i/ or /ɪ/.

| Modern English | Inglisce |
|----------------|----------|
| to sweep | to suipe |
| to swing | to suingue |
| twit | tuit |
| to tweet | to tuite |
| the tweet | þe tuite |
| tweets | tuíts |
| suite | * |
| swift | suift |
| swiss | suiss |
| Swedish | Suídish |

In this environment:
U = glide /w/  
I = vowel nucleus  

Plural or inflected forms may require **Ú** to preserve stress:
tuit → tuíts

---

# IV. Silent U Function (Hard G Preservation)

In some words, UI represents silent U + I.

The U preserves hard G before E, I, or Y  
(see U entry for full rule hierarchy).

| Modern English | Inglisce |
|----------------|----------|
| to guide | to guîde |
| guild | * |
| guilty | * |

Here:
GU = hard /g/  
U = silent  
I = vowel nucleus  

---

# V. Structural Summary

| Pattern | Value |
|----------|--------|
| ÚI (stressed) | /uɪ/ |
| UI (unstressed) | /wi/, /wɪ/ |
| GUI | hard G + /i/ |

---

# Teaching Notes

• Stressed UI must be written **ÚI**  
• Unstressed UI defaults to glide + vowel  
• UI after G preserves hard G  
• Stress marking prevents ambiguity between /uɪ/ and /wi/
