# UI / ÚI

The digraph **UI** represents different sounds depending on stress and position.

---

## I. Pronunciation Overview

| Form | Stress | Value |
|------|--------|--------|
| ÚI | Stressed | /uɪ/ |
| UI | Unstressed | /wi/ or /wɪ/ |
| -UIS | Unstressed | /juz/ |
| GUI | — | Hard G + /i/ (U silent) |

'*' Represents no spelling change from Modern (Websterian) English.

---

## II. Stressed ÚI → /uɪ/

When stressed, UI is written **ÚI** and pronounced /uɪ/.

| Modern English | Inglisce |
|----------------|----------|
| to ruin | to rúine |
| the ruin | þe rúin |

This spelling explicitly marks stress and prevents glide reduction.

---

## III. Unstressed UI → /w/ + vowel

When unstressed, **UI(E** represents a glide /w/ followed by /i/ or /ɪ/.

| Modern English | Inglisce |
|----------------|----------|
| suite | * |
| sweet | suiet |
| to sweep, swept | to suiepe |
| to swing | to suingue |
| the tweet, tweets | þe tuite, tuíts |
| Swedish | Suídish |
| twit | tuit |
| swift | suift |
| swiss | suiss |


In this environment:
U = glide /w/  
I = vowel nucleus  

---

## IV. Plural -UIS → /juz/

The spelling **-UIS** occurs **only in plural forms**.

It represents /juz/ and replaces Modern English -ues / -ews.

| Modern English | Inglisce |
|----------------|----------|
| value, values | value, valuis |
| curfew, curfews | corfue, corfuis |

This spelling is **morphological**, not phonetic:
There is no singular *-ui* form with this value.

---

## V. Silent U Function (Hard G Preservation)

In some words, UI represents silent U + I.

The U preserves hard G before E, I, or Y  
(see U entry for full rule hierarchy).

| Modern English | Inglisce |
|----------------|----------|
| to guide | to guîde |
| guild | * |
| guilty | * |

Here:
GU = hard /g/  
U = silent  
I = vowel nucleus  

---

## VI. Structural Summary

| Pattern | Value |
|----------|--------|
| ÚI (stressed) | /uɪ/ |
| UI (unstressed) | /wi/, /wɪ/ |
| -UIS | Unstressed | /juz/ |
| GUI | hard G + /i/ |

---

## Teaching Notes

• Stressed UI must be written **ÚI**  
• Unstressed UI defaults to glide + vowel  
• UI after G preserves hard G  
• Stress marking prevents ambiguity between /uɪ/ and /wi/
