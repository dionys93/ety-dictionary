# UE

The digraph **UE** represents three phonological values in Inglisce.

## Pronunciation Values

| Environment | Value |
|-------------|--------|
| Word-final | /u/ |
| Latin/French derivatives | /uə/ |
| ÜE | /wə/ |

'*' Represents no spelling change from Modern (Websterian) English.

---

# I. Word-Final UE → /u/

**UE** represents a long /u/ or /ju/ sound.
See yod rules in **U.md** for details.

| Modern English | Inglisce | IPA |
|----------------|----------| --- |
| blue, sue | * | /blu/, /su/ |
| value | * | /ˈvælju/ |

This is the stable long vowel realization.

---

# II. Latin/French Derivatives → /uə/

In polysyllabic words of Latin or French origin,  
**UE** represents the sequence **/uə/**.

This typically occurs in:
- -uent  
- -uence  

| Modern English | Inglisce |
|----------------|----------|
| congruent, congruence  | congrúent, congrúence |
| confluent, confluence | conflúent, conflúence |
| affluent, affluence | * |
| influence | * |


This reflects historical vowel retention in learned vocabulary.

---

# III. ÜE → /wə/

ÜE represents **/wə/**.

The glide is explicitly marked with **Ü** in Inglisce.

| Modern English | Inglisce |
|----------------|----------|
| sequent, sequence | siqüent, siqüence |
| consequent, consequence | conseqüent, conseqüence |
| subsequent, subsequence | sobseqüent, sobseqüence |
| eloquent, eloquence | eloqüent, eloqüence |

---

# IV. Structural Summary

| Pattern | Value |
|----------|--------|
| UE (final) | /u/ |
| UE (Latin/French medial) | /uə/ |
| ÜE | /wə/ |
