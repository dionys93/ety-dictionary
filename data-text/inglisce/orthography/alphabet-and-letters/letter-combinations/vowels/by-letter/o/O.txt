The letter O and its variants represent multiple vowel sounds in Inglisce.

O uses the following diacritics: Ò, Ó, Ô

Related digraphs: See OA, OI, OU, OY files

PRONUNCIATION SUMMARY:
O — /ɑ/ "lock" — Very common — lock → loc;
    /oʊ/ "home" — Very common — home;
    /ɔ/ "caught" (before H) — Common — bought → boht;
    /ə/ "common" (unstressed) — Very common — seven → sevon;
    /ər/ "doctor" (unstressed before R) — Common — doctor;
    /w/ "wet" — Uncommon — wet → ouet, white → oît
Ò — /ɔ/ "cost" (stressed) — Uncommon — cost → còste
Ó — /ɑ/ "father" (stressed) — Uncommon — phonology → fonólogie  
Ô — /oʊ/ "old" (stressed); /aʊ/ (before N/U); /ʊ/ (before C/D) — Common — old → ôld; sound → sônd; book → bôc

====================================

O — BASE LETTER

PRONUNCIATION

/ɑ/ — as in "lock" (before two consonants) [Very common]
/oʊ/ — as in "home" (various contexts) [Very common]
/ɔ/ — as in "bought" (before H) [Common]
/ə/ — as in "Monday" (unstressed) [Very common]
/ər/ — as in "doctor" (unstressed before R) [Common]
/w/ — as in "wet" [Uncommon]

RULES

O + Two Consonants = /ɑ/ [Very common]
When followed by two consonants, or is the only vowel, O is typically /ɑ/.

not, bot, lot
concoction → concoccion
to concoct → to concocte
knot → gnot
to block → to blocque
block → bloc
to lock → to locque
lock → loc

O = /oʊ/ in Multiple Contexts [Very common]

Before single consonant + silent E:
cove, stove, lone, code, prone, zone, cone, 
phone → fone
to explode, to corrode, to condone
to suppose → to sopose
to postpone → to poastpone
to oppose → to opose
throne → þrone

Word-final in multisyllables:
tomorrow → tomòro
tarot → târo
sparrow → spâro
window → uindo
show → seoe, blow → bloe, flow → floe

Stressed verb infinitives use -OE:
to lasso → to lassoe
to follow → to folloe
Gerunds: lassouing, fóllouing

O + H = /ɔ/ [Common]
OH represents /ɔ/, typically in irregular past tenses.

bought → boht
fought → foht
sought → çoht
thought → þoht

Unstressed O = /ə/ [Very common]
When unstressed, O is often /ə/.

monotheism → monothýism /ˌmɑnəˈθiˌɪzəm/
museum → m̃usíom /mjuˈziəm/
stadium → stâdiom /ˈsteɪdiəm/
run → rone /rən/
done → don
to come, have come → to come, have com
Monday /ˈmənˌdeɪ/

Note: O replaces U after M, N, R (U cannot appear in these positions)

Unstressed O + R = /ər/ [Common]
When unstressed and followed by R, O becomes /ər/.

turn → torne, return → retorne
administrator → administraitor /ədˈmɪnəˌstreɪɾər/
gopher → goafor /ˈɡoʊfər/
educator → eḑucaitor /ˈɛdʒəˌkeɪɾər/
hammer → hamor /ˈhæmər/
murmur → mormor
murder → mordre

O as /w/ [Uncommon]
In specific borrowings and patterns.

choir → coîre
why → oîe
white → oît
wet → ouet
penguin → pengoin
wet → ouet

Ò — STRESSED /ɔ/

Pronunciation: /ɔ/ — as in "cost" (always stressed)

PURPOSE
Marks stressed /ɔ/ where stress placement needs clarification.

EXAMPLES

cost → còste
frost → fròste
cough → còffe
tomorrow → tomòro

CROSS-REFERENCES

See IPA-ɔ.txt for all /ɔ/ patterns

====================================

Ó — STRESSED /ɑ/

Pronunciation: /ɑ/ — as in "bot" (always stressed)

PURPOSE
Marks stressed /ɑ/ in words where stress placement needs clarification.

EXAMPLES

Greek-derived terms:
anonymous → anónimus
phonology → fonólogie
phonological → fonológical

Latin terms:
contract → cóntract

CROSS-REFERENCES

Ô — MULTIPLE STRESSED SOUNDS

Pronunciation: /oʊ/, /aʊ/, /ʊ/ (always stressed) [Common]

RULES

Basic Ô = /oʊ/ [Common]
In most contexts, Ô represents stressed /oʊ/.

hypnosis → hipnôsis
bold → bôld, cold → côld, old → ôld, sold → sôld
gold → gôld, hold → hôld, mold → môld

Ô + N/U = /aʊ/ [Common]

sound → sônd
around → arônd
ground → grônd
cow → côue
vow → vôue
sow → sôue

Ô + C/D = /ʊ/ [Common]

book → bôc
cook → côc
look → lôc
hood → hôd
wood → uôd

After U, Ô replaces Û [Uncommon]
Special rule: Û cannot appear after U, use Ô instead.

woman → uôman
wolf → uôlfe

CROSS-REFERENCES

See IPA-oʊ.txt, IPA-aʊ.txt, IPA-ʊ.txt for respective sound patterns

TEACHING NOTES

O is highly variable based on context
Most common: stressed O before two consonants = /ɑ/
O + silent E usually = /oʊ/
Unstressed O typically = /ə/
O replaces U after M, N, R
Ô changes based on following letter: default /oʊ/, before N/U = /aʊ/, before C/D = /ʊ/
