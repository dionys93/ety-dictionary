The letter O and its variants represent multiple vowel sounds in Inglisce.

O uses the following diacritics: Ò, Ó, Ô

Related digraphs: See OA, OI, OU, OY files

PRONUNCIATION SUMMARY:
O — /ɑ/ "lock" — lock → loc;
    /oʊ/ "home" — home;
    /ɔ/ "caught" (before H) — bought → boht;
    /ə/ "Monday" (unstressed) — Monday, seven → sevon;
    /ər/ "doctor" (unstressed before R) — doctor;
    /w/ "wet" — wet → ouet, white → oît
Ò — /ɔ/ "cost" (stressed) — cost → còste
Ó — /ɑ/ "father" (stressed) — phonology → fonólogie  
Ô — /oʊ/ "old" (stressed); /aʊ/ (before N/U); /ʊ/ (before C/D) — old → ôld; sound → sônd; book → bôc

====================================

O + Two Consonants = /ɑ/ 
When followed by two consonants, or is the only vowel, O is typically /ɑ/.

not, bot, lot
concoction → concoccion
to concoct → to concocte
knot → gnot
to block → to blocque
block → bloc
to lock → to locque
lock → loc

O = /oʊ/

Before single consonant + silent E:
cove, stove, lone, code, prone, zone, cone 
phone → fone
to explode, to corrode, to condone
to suppose → to sopose
to postpone → to poastpone
throne → þrone
to oppose

Word-final in multisyllables:
tomorrow → tomòro
tarot → târo
sparrow → spâro
window → uindo
show → seoe, blow → bloe, flow → floe

Unstressed verb infinitives use -OE:
to lasso → to lassoe
to follow → to folloe
Gerunds: lassouing, follouing

O + H = /ɔ/ 
OH represents /ɔ/, typically in irregular past tenses.

bought → boht
fought → foht
sought → çoht
thought → þoht


Unstressed O = /ə/
When unstressed, O is often /ə/.

Regular unstressed O:
monotheism → monothýism /ˌmɑnəˈθiˌɪzəm/
museum → m̃usíom /mjuˈziəm/
stadium → stâdiom /ˈsteɪdiəm/
to come, have come → to come, have com

O replacing U after M/N/R:
O systematically replaces U after M, N, or R - a spelling rule where U is forbidden.
The pronunciation remains /ə/ as it would with U:

run → rone /rən/ 
done → don /dən/ 
come → com /kəm/ 
money → monie /ˈməni/ 
number → nombre /ˈnəmbər/ 
murmur → mormor /ˈmərˌmər/
Monday /ˈmənˌdeɪ/

CROSS-REFERENCE: See U.txt "Restriction: M/N/R Rule" for full explanation.


Unstressed O + R = /ər/ 
When unstressed and followed by R, O becomes /ər/.

turn → torne, return → retorne
administrator → administraitor /ədˈmɪnəˌstreɪɾər/
gopher → goafor /ˈɡoʊfər/
educator → eḑucaitor /ˈɛdʒəˌkeɪɾər/
hammer → hamor /ˈhæmər/
murmur → mormor
murder → mordre

O as /w/ 
In specific borrowings and patterns.

choir → coîre
why → oîe
white → oît
wet → ouet
penguin → pengoin
wet → ouet

Ò — STRESSED /ɔ/

Pronunciation: /ɔ/ — as in "cost" (always stressed)

EXAMPLES

cost → còste
frost → fròste
cough → còffe
tomorrow → tomòro

CROSS-REFERENCES

See IPA-ɔ.txt for all /ɔ/ patterns

====================================

Ó — STRESSED /ɑ/

Pronunciation: /ɑ/ — as in "bot" (always stressed)

PURPOSE
Marks stressed /ɑ/ in words where stress placement needs clarification.

EXAMPLES

Greek-derived terms:
anonymous → anónimus
phonology → fonólogie
phonological → fonológical

Latin terms:
contract → cóntract

CROSS-REFERENCES

See IPA-ɑ.txt for all /ɑ/ patterns
See also Á.txt (stressed /ɑ/ in basic words)

====================================

Ô — MULTIPLE STRESSED SOUNDS

Pronunciation: /oʊ/, /aʊ/, /ʊ/ (always stressed) 

RULES

Basic Ô = /oʊ/ 
In most contexts, Ô represents stressed /oʊ/.

hypnosis → hipnôsis
bold → bôld, cold → côld, old → ôld, sold → sôld
gold → gôld, hold → hôld, mold → môld

Ô + N/U = /aʊ/

sound → sônd
around → arônd
ground → grônd
cow → côue
vow → vôue
sow → sôue

Ô + C/D = /ʊ/

book → bôc
cook → côc
look → lôc
hood → hôd
wood → uôd

After U, Ô replaces Û
Special rule: Û cannot appear after U, use Ô instead.

woman → uôman
wolf → uôlfe

CROSS-REFERENCES

See IPA-oʊ.txt, IPA-aʊ.txt, IPA-ʊ.txt for respective sound patterns

TEACHING NOTES

O is highly variable based on context
Most common: stressed O before two consonants = /ɑ/
O + silent E usually = /oʊ/
Unstressed O typically = /ə/
O replaces U after M, N, R
Ô changes based on following letter: default /oʊ/, before N/U = /aʊ/, before C/D = /ʊ/
