# IPA /ʊ/ — "BOOK" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /ʊ/ |
| Description | Near-high near-back rounded vowel |
| Example in Modern English | book, good, could |
| Frequency | Moderately common; often in high-frequency words |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | Ô (stressed) |
| Secondary | Û (stressed) |
| Modal verbs | OÛ |
| Affricate cluster | -USCE |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

---

## Core Distribution Rules

### 1. Ô = /ʊ/ (Stressed)

The default spelling for /ʊ/ in stressed syllables, typically corresponding to Modern English OO or O.

#### Singular / Plural Pairs

Note: plurals of ÔC words follow an irregular -EOX pattern.

| Modern | Inglisce |
|--------|----------|
| book, books | bôc, beox |
| look, looks | lôc, leox |
| hook, hooks | hôc, heox |
| cook, cooks | côc, queox |
| hood, hoods | hôde, hôds |
| wood, woods | uôde, uôds |
| woman, women | uôman, uimmen |
| wolf, wolves | uôlfe, uôlfs |
| wool | uôle |
| soot | sôte |
| foot, feet | fôte, fyt |

---

### 2. OÛ = /ʊ/ (Modal Verbs)

Used exclusively in the core modal verbs where Modern English has OU or OUL.

| Modern | Inglisce |
|--------|----------|
| could | coûd |
| would | oûd |
| should | seûd |

---

### 3. Û = /ʊ/ (Stressed)

Used in shorter, high-frequency words and closed syllables not covered by Ô.

| Modern | Inglisce |
|--------|----------|
| good | gûd |
| full | fûl |
| cushion | cûcion |
| to pull | to pûle |
| to put | to pûte |

---

### 4. -USCE = /ʊʃ/ (Affricate Cluster)

Used for words ending in Modern English -USH.

| Modern | Inglisce |
|--------|----------|
| bush | busce |
| push | pusce |
