# IPA /u/ — "MOON" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /u/ |
| Description | High-back rounded vowel |
| Example in Modern English | moon, blue, food |
| Frequency | Common; appears in multiple spelling patterns |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | OU |
| Silent E | U-E (yod class = /ju/; non-yod class = /u/) |
| Word-final | UE |
| Post-R / Post-TR | EU |
| Stress-marked | Ú |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

---

## Core Distribution Rules

### 2. U-E = /u/ or /ju/ (Silent E)

Long U restored by silent E. Pronunciation depends on yod class of the preceding consonant.

#### Yod Class (preceding c, g, p, b, f, h, v, m, j) → /ju/

| Modern | Inglisce |
|--------|----------|
| cute | * |
| mute | * |
| muse | * |

#### Non-Yod Class (preceding t, d, s, n, z, r) → /u/

| Modern | Inglisce |
|--------|----------|
| rude | * |
| crude | * |

#### Before Vowels → /u/

| Modern | Inglisce |
|--------|----------|
| nuance | * |
| congruence | * |
| influence | * |

---

### 3. UE = /u/ (Word-Final)

Used in word-final position corresponding to Modern English -UE, -IEW, and -UE.

| Modern | Inglisce |
|--------|----------|
| blue | bleu |
| clue | cleue, cleus |
| avenue | avenoue, avenous |
| due, dues | dûe, dûs |
| to sue | * |
| view | vue, vuis |
| hue, hues | hue, huis |

---

### 4. EU = /u/ (After R, TR)

Used after R and TR, where the non-yod class produces plain /u/ without the OU spelling.

| Modern | Inglisce |
|--------|----------|
| rule, rules | reule, reuls |
| true | treu |
| truth | treuþ |
