IPA /ə/ — "SCHWA" VOWEL

SOUND OVERVIEW
- IPA symbol: /ə/
- Description: Mid-central unrounded vowel
- Audio equivalent: as in "about", "taken", "common"
- Frequency: Extremely common in Inglisce, the default unstressed vowel

SPELLING PATTERNS
- Primary: O, A, U (in specific contexts)
- Secondary: I
- Contextual: U between certain consonants

DETAILED RULES

1. O = /ə/ (Unstressed)
   - Unstressed positions: monotheism → monoþÿism, museum → m̃usíom
   - Replaces U after M, N, R: run → rone, Monday, nun → nonne
     suppose → sopose, condition → condicion, justice → jostice
   - Administrator: administrator → administraitor

2. A = /ə/ (Unstressed)
   - Unstressed syllables: about → abaut, mother → maþre
   - Variable stress words: animate (adj.) → animat 
   - Common prefixes: practical, logical

3. U = /ə/ 

cut → cutte, shut → seute
but
nut, nuts → nutte, nuts
gut, guts → gutte, guts

4. I = /ə/ (Unstressed)
   - Unstressed positions: individual → indiviḑual
