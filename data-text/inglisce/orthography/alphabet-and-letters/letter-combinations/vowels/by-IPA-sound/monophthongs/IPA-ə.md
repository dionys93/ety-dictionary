# IPA /ə/ — "SCHWA" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /ə/ |
| Description | Mid-central unrounded vowel |
| Example in Modern English | about, taken, common |
| Frequency | Extremely common; the default unstressed vowel |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | O, A, U (in specific contexts) |
| Secondary | I (unstressed) |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

---

## Core Distribution Rules

### 1. O = /ə/ (Unstressed)

O is the most common spelling for /ə/, appearing in unstressed syllables across a wide range of word types.


| Modern | Inglisce |
|--------|----------|
| Monday | * |
| run | rone |
| nun | nonne |
| suppose | sopose |
| condition | condicion |
| justice | jostice |
| monotheism | monoþÿism |
| museum | m̃usíom |
| administrator | administraitor |

---

### 2. A = /ə/ (Unstressed)

A represents /ə/ in unstressed syllables, common prefixes, and adjective forms reduced from fuller vowels.


| Modern | Inglisce |
|--------|----------|
| about | abaut |
| mother | maþre |
| animate (adj.) | animat |
| practical, logical | * |

---

### 3. U = /ə/ (Unstressed)

U represents /ə/ in short stressed monosyllables and common function words.

#### Words with Changes

| Modern | Inglisce |
|--------|----------|
| but | * |
| cut | cutte |
| shut | seute |
| nut, nuts | nutte, nuts |
| gut, guts | gutte, guts |

---

### 4. I = /ə/ (Unstressed)

I reduces to /ə/ in unstressed syllables, particularly in longer Latinate words.

| Modern | Inglisce |
|--------|----------|
| individual | indiviḑual |
