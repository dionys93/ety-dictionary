# IPA /i/ — "MACHINE" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /i/ |
| Description | High front unrounded vowel |
| Example in Modern English | machine, see, field |
| Frequency | Very common; appears across multiple spelling environments |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | I-E |
| Alternative | IE, EI- |
| Diacritic | Ï + vowel, ÿ + vowel |
| Affricate cluster | -IEC̃E, -OIC̃E |
| Stress-marked | Í |
| Secondary | Y (word-final adjectives/adverbs only) |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

---

## Core Distribution Rules

### 1. I-E = /i/ (Silent E)

The default spelling for /i/ in closed syllables and verb stems.

#### Verbs

| Modern | Inglisce |
|--------|----------|
| to see | to sihe |
| to compete | to compite |
| to repeat | to repite |
| to defeat | to defite |
| to complete | to complite |
| to repeal | to repile |
| to receive | to recive |
| to conceive | to concive |
| to deceive | to decive |

|--------|----------|

### 2. IE = /i/

Used in word-final syllables and within stems where Modern English uses EE or EA.

#### Word-Final (from Modern -E endings)

| Modern | Inglisce |
|--------|----------|
| committee, committees | comittie, comittis |
| recipe, recipes | recipie, recipis |
| catastrophe | catàstrofie |
| anemone, anemones | anèmonie, anèmonis |
| hyperbole, hyperboles | hypêrbolie, hypêrbolis |

#### Word-Final (from Modern -Y endings)

| Modern | Inglisce |
|--------|----------|
| enemy, enemies | enemie, enemis |
| policy, policies | policie, policis |
| community, communities | comunetie, comunetis |

#### Verbs 

| Modern | Inglisce |
|--------|----------|
| to feel | to fiele |

### 3. EI- = /i/ (Word-Initial)

| Modern | Inglisce |
|--------|----------|
| each | eic̃e |
| eagle | eigle |
| ease | eise |

---

### 4. Ï + Vowel = /i/

Diaeresis marks /i/ when it precedes another vowel within a syllable.

| Modern | Inglisce |
|--------|----------|
| eel | iale |
| seal | sïale |

---

### 5. ÿ + Vowel = /i/

Used in verb forms where the /i/ stem vowel precedes the -ING suffix.

| Modern | Inglisce |
|--------|----------|
| agreeing | agrÿing |

---

### 6. -IEC̃E / -OIC̃E = /iːtʃ/

Used for words ending in Modern English -EACH or -EECH. 

#### -IEC̃E

| Modern | Inglisce |
|--------|----------|
| beach | biec̃e |
| leach | liec̃e |
| impeach | impiec̃e |
| leak | lieche |

#### -OIC̃E

| Modern | Inglisce |
|--------|----------|
| beech | boic̃e |
| leech | loic̃e |
| peach | poic̃e |

---

### 7. Í = /i/ (Stress-Marked)

Acute accent marks stressed /i/ in derived and polysyllabic forms.

#### Derived Forms (-TION, -MENT)

| Modern | Inglisce |
|--------|----------|
| completion | complícion |
| deletion | delícion |
| agreement | agríment |

#### Receipt / Conceit / Deceit Pattern

| Modern | Inglisce |
|--------|----------|
| receipt, receipts | recite, recípts |
| conceit, conceits | concite, concípts |
| deceit, deceits | decite, decípts |

#### Other Stressed Forms

| Modern | Inglisce |
|--------|----------|
| immediate | immídiat |
| media | mídia |
| idea | îdía |

---

### 8. Y = /i/ (Word-Final Adjectives and Adverbs Only)

Used exclusively in word-final position for adjectives and their adverbial derivatives.

| Modern | Inglisce |
|--------|----------|
| happy, happily | * |
| silly | * |
| funny | fonny |
| pretty | pritty |
| quickly | cuicly |
| slowly | sloaly |
