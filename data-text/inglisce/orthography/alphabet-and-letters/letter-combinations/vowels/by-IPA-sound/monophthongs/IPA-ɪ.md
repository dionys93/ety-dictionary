# IPA /ɪ/ — "BIT" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /ɪ/ |
| Description | Near-high near-front unrounded vowel |
| Example in Modern English | bit, sit, condition |
| Frequency | Very common; especially in stressed monosyllables |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | I (stressed, before consonants) |
| Diacritic | Ï (separator and stress indicator) |
| Stress-marked | Ÿ (Y-based) |
| Contextual | I in -ION nouns |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

---

## Core Distribution Rules

### 1. I = /ɪ/ (Stressed Monosyllables)

The default spelling for /ɪ/ in stressed monosyllables before stop consonants.

| Modern | Inglisce |
|--------|----------|
| bit, bits | bitte, bits |
| hit, hits | hitte, hits |
| kit, kits | chite, chits |
| zit, zits | zitte, zits |
| pit, pits | pitte, pits |
| mit, mits | mitte, mits |

---

### 2. I = /ɪ/ (Before Consonant Clusters)

Used before C, CK, CH, and other clusters.

| Modern | Inglisce |
|--------|----------|
| sick | sic |
| rich | ric̃e |
| ditch | dic̃e |
| quick | cuic |
| thick | þic |
| to commit | to comitte |
| to permit | to permitte |
| to convict | to convicte |
| citizen | citasan |
| to knit | to gnite |

---

### 3. I = /ɪ/ (Stressed in -ION Nouns)

I retains /ɪ/ in stressed syllables before -ION endings.

| Modern | Inglisce |
|--------|----------|
| condition | condicion |
| ambition | ambicion |
| conviction | conviccion |
| fiction | ficcion |
| addition | addicion |
| addiction | adiccion |
| commission | comission |
| permission | * |

---

### 4. Ï / Ÿ = /ɪ/ (Diacritic Stress and Separation)

The diaeresis serves two functions: separating adjacent vowels to prevent misreading, and marking stressed /ɪ/ in irregular positions. Ï is used on I; Ÿ is used when the underlying letter is Y.

| Modern | Inglisce | Notes |
|--------|----------|-------|
| bid | bïde | Irregular verb form; ï prevents misread as /i/ |
| to nauseate | to nausïait | Specifies SI = /zi/, not /ʒ/ |
| image | ïmage | Word-initial separation |
| equipment | equïpment | Medial separation |
| anonymity | anonimatie | |
| considerable | considarable | |
| certificate | certificat | |
| inimical | * | |
| analytical | analÿtical | Y-based stress marking |
