# IPA /ɛr/ — "SQUARE" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /ɛr/ |
| Description | Mid-front unrounded vowel + rhotacization |
| Example in Modern English | square, care, bear |
| Frequency | Common; appears in verbs and derivatives |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | -AIRE, -EIRE |
| Stressed derivatives | -AÎR, -EÎR |
| Circumflex | -ÂRE |
| Word-final | -EARE |
| Marked stress | ÈR |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

### 3. -AÎR / -EÎR = /ɛr/ (Stressed Derivatives)

Circumflex î marks stressed /ɛr/ in derivatives of -AIRE and -EIRE stems.

| Modern | Inglisce |
|--------|----------|
| comparison | compaîrasson |
| comparative | compaîratif |
| comparable | compaîrable |
| unbearable | unbeîrable |

---

### 4. -ÂRE = /ɛr/

Used in words derived from Latin or French where Modern English has -ARE.

| Modern | Inglisce |
|--------|----------|
| square | squâre |
| dare | dâre |
| fare | fâre |
| hare | hâre |
| to prepare | to prepâre |
| to repair | to repâre |
| to glare | to glâre |
| to stare | to stâre |

---

### 5. -EARE = /ɛr/ (Word-Final)

Used in a small set of word-final forms corresponding to Modern English -EAR.

| Modern | Inglisce |
|--------|----------|
| bear, bears | beare, bears |
| pear, pears | peare, pears |
| to share | to seare |

---

### 6. ÈR = /ɛr/ (Marked Stress)

Grave accent marks stressed /ɛr/ where stress is not predictable from spelling alone.

| Modern | Inglisce |
|--------|----------|
| era | èra |
| to inherit | to inhèrite |
| to bury | to bèrie |
| burial | bèrial |
