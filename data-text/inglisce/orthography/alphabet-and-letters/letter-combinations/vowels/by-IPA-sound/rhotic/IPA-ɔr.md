# IPA /ɔr/ — "NORTH" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /ɔr/ |
| Description | Mid-back rounded vowel + rhotacization |
| Example in Modern English | north, war, warm |
| Frequency | Moderately common |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | -OR, -ÓR (stressed) |
| Secondary | -AUR |
| U-initial | U-, GU- (irregular), ÜA- |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

---

## Core Distribution Rules

### 1. -ORE / -ÓRE = /ɔr/ (Stressed)

-OR- is inherently stressed when followed by a consonant.

#### Monosyllables (stress inherent)

| Modern | Inglisce |
|--------|----------|
| corn | corne |
| form | forme |
| storm | storme |
| to conform | to conforme |
| north | norþ |

#### Multisyllabic Verbs

-ORE is not inherently stressed in multi-syllable verbs, so an acute is required when it is.

| Modern | Inglisce |
|--------|----------|
| to ignore | to ignóre |
| to abhor | to abhóre |
| to hammer | to hamore (stressed on first syllable) |
| to rumor | to rúmore (stressed on first syllable) |

---

### 2. -AUR = /ɔr/

| Modern | Inglisce |
|--------|----------|
| roar, roars | raure, raurs |
| boar, boars | baure, baurs |
| oar, oars | oare, oars |

---

### 3. U- = /ɔr/ (W-Initial Words)

| Modern | Inglisce |
|--------|----------|
| warm | uarm |
| warn | uarne |
| warden | uardan |
| swarm | suarme |

---

### 4. ÜA- = /ɔr/ (Ü Glide Marker)

Ü marks the /w/ glide in order to remove ambiguity.

| Modern | Inglisce |
|--------|----------|
| award | aüard |
| reward | reüard |

---

### 5. GU- = /ɔr/ (Irregular)

| Modern | Inglisce |
|--------|----------|
| war | guare |
| warrior | guarior |
