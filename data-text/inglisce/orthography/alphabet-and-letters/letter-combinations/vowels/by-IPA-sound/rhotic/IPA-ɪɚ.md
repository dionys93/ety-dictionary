# IPA /ɪər/ — "NEAR" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /ɪər/ |
| Description | Near-high near-front unrounded vowel + rhotacization |
| Example in Modern English | near, here, beer |
| Frequency | Moderately common |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | -IOR, -IAR, -IER |
| Secondary | -IERE |
| Noun and Verb Form | -IRE |
| Rare | -YRE |

'*' Represents no change in spelling between Modern (Websterian) English and Inglisce

---

## Core Distribution Rules

### 1. -IOR = /ɪər/ 

Used where Modern English has -EER.

| Modern | Inglisce |
|--------|----------|
| beer | bior |
| deer | dior |
| to leer | to liore -s -d lioring |
| leer | lior |
| leery | liory, lioriar, lioriest |
| steer | stior |

---

### 2. -IAR = /ɪər/

| Modern | Inglisce |
|--------|----------|
| dear | diar |
| fear | fiar |
| to fear | to fiare -s -d fiaring |
| ear | iar |
| ear (of corn) | ihar |
| gear, gears | guire, guiars |
| to volunteer | to volontier, volontire(s, -d, volontiering |
| volunteer, volunteers | volontire, volontiers |
| to rear (offspring) | to riare -s -d -ing |

### 3. -IER, -IRE = /ɪər/

| Modern | Inglisce |
|--------|----------|
| to near | to nier nire(s nired niering |

| to volunteer | to volontier, volontire(s, -d, volontiering |

| gear, gears | guire, guiars |

---

### 4. -IERE = /ɪər/

| Modern | Inglisce |
|--------|----------|
| here | hiere |
| to cheer | to c̃iere -s -d c̃iering |
| cheer, cheers | c̃iere, c̃iers |
| to steer | to stiere -s -d stiering |
| rear (behind) | riere, riers |
| to rear (be behind) | to riere -s -d riering |

---

### 5. -YRE = /ɪər/

H triggers a spelling shift from -IRE to -YRE.

| Modern | Inglisce |
|--------|----------|
| to hear | to hyre |
