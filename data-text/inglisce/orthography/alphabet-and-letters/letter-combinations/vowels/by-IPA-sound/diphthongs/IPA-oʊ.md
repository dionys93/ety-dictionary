# IPA /oʊ/ — "GOAT" DIPHTHONG

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /oʊ/ |
| Description | Mid-back to high-back closing diphthong |
| Example in Modern English | goat, home, show |
| Frequency | Very common; appears across multiple spelling environments |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | O-E |
| Alternative | OA |
| Word-final | -OE, -O |
| Stress-marked | Ó |

---

## Core Distribution Rules

### 1. O + Single Consonant + Silent E = /oʊ/

This is the default closed-syllable spelling.

#### No Changes from Modern to Inglisce:
