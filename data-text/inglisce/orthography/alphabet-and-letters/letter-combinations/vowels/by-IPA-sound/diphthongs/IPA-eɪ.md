# IPA /eɪ/ — "MAKE" VOWEL

## Sound Overview

| Feature | Value |
|----------|--------|
| IPA | /eɪ/ |
| Description | Mid-front closing diphthong |
| Example words (Modern English) | make, day, rain |
| Frequency | Very common; appears in many word types and positions |

---

## Spelling Patterns

| Category | Spellings |
|-----------|------------|
| Primary | AI-E, AY, Â |
| Secondary | EI, EY, EIH |
| Stress-marked | ÉI, Â-E |
| Positional | -AIG (before N) |

---

## Core Distribution Rules

### 1. AI = /eɪ/ (Verbs & Mid-Word)

| Modern | Inglisce |
|--------|----------|
| to say | to saie |
| to pay | to paie |
| to stay | to staie |
| rain | raigne |
| stain | staigne |
| complaint | complaignte |
| champagne | ciampaigne |

#### Verb Paradigm Examples

| Base | Gerund |
|-------|----------|
| to saie | saying |
| to paie | paying |

#### Simple Nouns

| Modern | Inglisce | 
|---------|-----------|
| day, days | daie, days |
| bay, bays | baie, bays |
| ray, rays | raie, rays |
| play, plays | plaie, plays |

### 4. EY = /eɪ/ (Word-Final)

| Category | Examples |
|------------|------------|
| Pronouns | they → þey |
| Adjectives | whey → ouey |

### 5. Latin -ATE Verbs (Unstressed /eɪ/)

Most Latin -ate verbs use **-AIT** when unstressed.

| Modern | Inglisce |
|----------|-----------|
| to demonstrate | to demonstrait |
| to gravitate | to gravitait |
| to isolate | to îsolait |
| to elevate | to elevait |
| to complicate | to complicait |
| to navigate | to navigait |

#### Stressed Exceptions use Â

| Modern | Inglisce |
|----------|-----------|
| to relate | to relâte |
| to create | to criâte |
| to negate | to negâte |
