# IPA /aɪ/ — "EYE" DIPHTHONG

The phoneme /aɪ/ is represented by several orthographic strategies in Inglisce.
Distribution depends on position, morphology, and historical class.

---

## 1. Core Marker: î

Primary and most productive spelling.

| Modern English | Inglisce | Part of Speech |
|----------------|----------|-------|
| knight, knights | nîhte, nîhts | noun |
| sight, sights | sîhte, sîhts | noun |
| cry, cries | crîe, crîs | noun |
| fry, fries | frîe, frîs | noun |
| try, tries | trîe, trîs | noun |
| dry | drîe | verb |
| dice | dîce | noun/verb |
| isle, isles | île, îals | noun |
| spine, spines | spîne, spîns | noun |
| spider, spiders | spîdre, spîdars | noun |
| to bind | to bînde | verb |
| to find | to fînde | verb |
| to remind | to remînde | verb |
| polite | polîte | adjective |
| quite | quît | adverb |

### Teaching Note
î is the default representation of /aɪ/.
Use unless another historical class applies.

---

## 2. YH Marker

Used in certain verbal and participial environments.

| Modern English | Inglisce |
|----------------|----------|
| dye | dyhe |
| tie, tight | tyhe, tyht |
| crying | cryhing |
| drying | dryhing |
| frying | fryhing |
| trying | tryhing |

### Function
Marks glide + morphological extension in -ing and related forms.

---

## 3. EA / AI Class (Lexical Set)

Certain lexical items preserve older vowel classes.

| Modern English | Inglisce |
|----------------|----------|
| eye, eyes | eaie, eais |
| aisle, aisles | aiele, aiels |

---

## 4. AE / ÁI Forms

Lexically restricted or semantic contrasts.

| Modern English | Inglisce |
|----------------|----------|
| fry, fries (small fish) | frae, fráis |
| chai, chais | c̃ae, c̃ais |
| to lie (recline) | to láie |

### Note
These forms often distinguish homophones or semantic categories.

---

## 6. -AIH Class
