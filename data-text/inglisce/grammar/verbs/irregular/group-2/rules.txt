GROUP 2 IRREGULAR VERBS: CONJUGATION PATTERNS

These verbs don't follow the regular "-ed" pattern for past tense and participle forms.

Pattern 1:  -ie → -e{c}t
Present → Past/Participle: 
  Examples: 
    fiele → felt
    sliepe → slept
    kiepe → kept
    ɯiepe → ɯept
    diele → delt
    miene → ment
    niele → nelt
    criepe → crept
    sɯiepe → sɯept
    lievfe → left


Pattern 2: Vowel Change in Past Tense
"i" → "a/o" pattern:
  Examples: 
    sitte → sate
    spitte → spate
    ɯinne → ɯon
    spinne → spon

"î" → "áu" pattern:
  Examples: 
    fînd → fáund
    grînd → gráund
    ɯînd → ɯáund
    bînd → báund


Pattern 3: "-oht" Pattern
Examples:
  byhe → boht
  bringue → broht
  tieche → toht
  sike → soht
  þinke → þoht
  fyhte → foht


Pattern 4: -{c}e → t
Examples:
  bilde → bilt
  spende → spent
  sende → sent
  lende → lent
  bende → bent
  smelle → smelt
  spille → spilt


SPECIAL CASES

Stand and Understand
Both "stand" and "understand" have a unique pattern:
ta stande -s stûd -ing
ta ondrestande -s ondrestûd -ing

Verbs with Different Forms for Transitive vs. Intransitive Uses
Some verbs have different forms depending on whether they're used transitively or intransitively:

Examples:
ta hângue -s hong -ing (intransitive)
ta hângue -s -d -ing (transitive)

ta criepe -s crept -ing (intransitive)
ta criepe -s -d -ing (transitive/intransitive)

ta boûne -s boûnt -ing (intransitive)
ta bêne -s -d -ing (transitive)

ta smelle -s smelt -ing (intransitive)
ta smelle -s -d -ing (transitive)

ta spille -s spilt -ing (intransitive)
ta spille -s -d -ing (transitive)

Some verbs change both their vowel and ending in past tense:
Examples:
  telle → told
  selle → sold
  shîne → shone
  seute → shót
